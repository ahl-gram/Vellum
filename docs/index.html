VELLUM

an atelier of imaginary cartography

Vellum surveys worlds that don't exist and drafts them as atlas charts. Give it a seed; it invents an island, simulates the rain that carves its rivers, grows its forests, founds its harbor towns, names everything in one of four invented languages — then sits down at the drafting table and draws the map. Same seed, same world, byte-identical chart.

How It Works

Vellum is a zero-dependency TypeScript engine. Terrain comes from domain-warped gradient noise; priority-flood hydrology guarantees every river reaches the sea; a Whittaker-style climate model grows the biomes; settlements score harbors and river mouths; Dijkstra roads share corridors; realm borders follow ridges and rivers. Names come from syllable grammars for four invented cultures. The whole pipeline is deterministic — fork-labeled RNG streams mean the same seed always draws the same world.

Draw your own: git clone https://github.com/ahl-gram/Vellum then npm run chart -- --seed 42 — or read the README.
